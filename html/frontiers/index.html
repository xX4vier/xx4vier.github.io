Um programa simples em C

Abaixo está um programa básico em C. Este programa C exibe "Hello World!" Na janela de saída. E todas as sintaxes e comandos na programação C diferenciam maiúsculas de minúsculas. Além disso, cada instrução deve ser finalizada com ponto-e-vírgula (;), que é um terminador de instrução.

           
            #include  <stdio.h>
int main()
{
   
   printf("Hello World! ");
   getch();
   return 0;
}
